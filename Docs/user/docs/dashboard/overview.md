# Dashboard

The dashboard is your salon command center. See everything at a glance when you log in.

---

## Today at a Glance

The top bar shows real-time numbers:

| Widget | Shows |
|---|---|
| 📅 **Today's Appointments** | Total booked vs. completed |
| 🚶 **Walk-ins** | Unscheduled clients today |
| 💰 **Revenue Today** | Total including tips (updates after each checkout) |
| 👤 **Active Clients** | Clients currently in chairs |
| ⭐ **Avg Rating** | Today's client satisfaction (if using feedback) |

---

## Calendar Mini-View

Left side of the dashboard shows today's appointments in a compact list:

- 🟢 = Checked in
- 🟡 = Waiting
- 🔵 = In service
- ⚪ = Booked but not arrived
- ✅ = Completed
- ❌ = No-show / cancelled

Click any appointment to open the full detail panel.

---

---

## Upcoming Appointments

Scrollable list of the next 48 hours:

- Client name, service, stylist, time
- Colored dots match service category colors
- Icons show if they booked online 🌐 or by phone 📞

---

## Quick Actions

Buttons for the most common tasks:

| Action | Shortcut |
|---|---|
| ➕ New Appointment | `N` key |
| 💰 Quick Checkout | `C` key |
| 👤 Add Client | `Ctrl+N` |
| 🔍 Search | `/` key |

---

## Revenue Chart

Bottom of the dashboard shows a mini bar chart:

- This week vs. last week
- Today highlighted
- Hover for exact numbers

[Full reports →](../reports/daily.md)

## Notifications

Bell icon in the top right shows:

- 🔔 Online booking received
- ⏰ Appointment reminders
- 📦 Low stock alerts
- 💬 Client messages

---

## Customizing Your Dashboard

Drag widgets to reorder. Pin the ones you use most. Go to **Settings → Dashboard** to toggle what shows.

!!! tip "Pro tip"
    If you're front desk, keep the calendar front and center. If you're the owner, pin revenue and staff overview.
