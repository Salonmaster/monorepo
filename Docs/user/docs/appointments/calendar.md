# Calendar View

The calendar is where you manage your salon's day. Book, move, cancel — everything happens here.

---

## Layout

The main calendar shows:

- **Columns** = Staff members (or rooms/stations)
- **Rows** = Time slots (configurable: 15, 30, or 60 min)
- **Blocks** = Appointments, colored by service category
- **Gaps** = Available time

Switch views:
- **Day** — today (default)
- **3-Day** — today + next 2 days
- **Week** — Monday–Sunday
- **List** — compact text list (good for printing)

Navigate with arrow keys or the date picker.

---

## Booking an Appointment

### From the Calendar

1. Click an empty time slot on the desired staff column
2. Search for an existing client or click **New Client**
3. Select one or more services
4. Adjust time/duration if needed
5. Add notes (optional)
6. Click **Book**

!!! tip "Quick book with keyboard"
    Press `N` → type client name → select service → Enter. Under 5 seconds.

### From the Sidebar

Click **+ New Appointment** in the sidebar for the full booking form with:
- Client selection
- Service selection with price preview
- Staff assignment (auto-suggests available)
- Date/time picker
- Recurring option (every 4 weeks, etc.)
- Notes field

## Appointment Statuses

| Status | Color | Meaning |
|---|---|---|
| **Booked** | ⚪ White | Appointment exists, client hasn't arrived |
| **Confirmed** | 🟣 Purple | Client confirmed (via SMS/email) |
| **Checked In** | 🟢 Green | Client is here, waiting |
| **In Service** | 🔵 Blue | Stylist has started |
| **Completed** | ✅ Checkmark | Done, ready for checkout |
| **No Show** | ❌ Red | Didn't arrive |
| **Cancelled** | 🗑 Grey | Cancelled in advance |

Change status by clicking the appointment → **Status** dropdown, or right-click the block.

---

## Moving & Editing

- **Drag & drop** — move to different time or staff member
- **Resize edges** — change duration
- **Double-click** — open full edit panel
- **Right-click** — context menu (check in, cancel, no-show, print)

Changes sync instantly across all devices. If reception moves an appointment, the stylist sees it immediately.

## Filters

Use the filter bar above the calendar:

- **Staff** — show/hide specific staff columns
- **Service** — only show certain services
- **Status** — e.g., only show checked-in clients
- **Client** — find a specific client's appointments

---

## Blocking Time

Need to block a staff member's time? Right-click → **Block Time**:

- **Break** — lunch, personal time
- **Meeting** — staff meeting
- **Training** — education block
- **Out of Salon** — offsite

Blocked time shows as grey and isn't bookable.

---

## Recurring Appointments

For clients who come every 4 weeks like clockwork:

1. Book the first appointment normally
2. Check **Recurring**
3. Set frequency (every 1–12 weeks)
4. Set end date or number of occurrences

SalonMaster auto-creates future appointments. They show with a 🔄 icon.
